Bahasa Utama Website (For Information)

Html (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk membangun struktur dasar dari sebuah halaman web.HTML bertindak sebagai fondasi dari semua elemen visual di situs web.

Css (Cascading Style Sheets)

CSS adalah bahasa yang digunakan untuk menata tampilan dari elemen-elemen HTML di halaman web. Dengan CSS, kita bisa mengatur warna, ukuran font, tata letak, dan animasi dari elemen-elemen pada halaman.

JS (JavaScript)

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas dan fungsi dinamis ke situs web. Dengan JavaScript, kita bisa membuat konten berubah secara otomatis, menangani input pengguna, berkomunikasi dengan server, dan banyak lagi.
